Sửa lỗi VMware Workstation Pro Can’t Run on Windows: Một trong những thông báo lỗi mà bạn có thể gặp khi sử dụng VMware Workstation Pro là thông báo lỗi VMware Workstation Pro Can’t Run on Windows. Thông báo lỗi cho biết phiên bản bạn đang chạy trên hệ thống của mình không tương thích với Windows, điều này có thể hơi vô lý khi bạn đang chạy nó tốt trước đó.

Sửa lỗi VMware Workstation Pro Can’t Run on Windows

Công cụ Program Compatibility Assistant trên Windows, được sử dụng để tìm kiếm bất kỳ vấn đề tương thích nào trên các chương trình bạn chạy trên hệ điều hành của mình, ngăn không cho VMware Workstation Pro 14 hoặc các phiên bản cũ hơn chạy trên hệ thống. Trong một số trường hợp, sự cố thậm chí có thể gặp phải trên Workstation Pro 15 nhưng đó là về nó. Hãy thử 6 cách sửa lỗi VMware Workstation Pro Can’t Run on Windows.

CÓ THỂ BẠN QUAN TÂM:
– VMware Workstation và Device/Credential Guard không tương thích
– Hướng dẫn cách bật TPM trong Hyper-V để cài đặt Windows 11
– Hướng dẫn sửa lỗi 0x800f080c khi bật Hyper-V trên Windows 10

# Đổi tên file thực thi VMware Workstation Pro

  • Trước hết, hãy mở cửa sổ File Explorer.
  • Sau đó, điều hướng đến thư mục đã cài đặt VMware Workstation Pro. Theo mặc định, bạn sẽ tìm thấy nó trong thư mục VMware nằm bên trong thư mục Program Files (x86) trong ổ đĩa cài đặt Windows.
  • Nhấp chuột phải vào vmware.exe và chọn đổi tên bạn muốn đổi.
  • Sau khi bạn đã làm điều đó, hãy tiếp tục và thử khởi chạy nó. Trong trường hợp nó không hoạt động, hãy khởi động lại máy tính của bạn xem đã sửa lỗi VMware Workstation Pro Can’t Run on Windows.

# Gỡ cài đặt các bản cập nhật Windows cụ thể

  • Nhấn phím Windows + I. Sau đó, trên Windows Settings, hãy thực hiện theo cách của bạn để Update and Security.
  • Tại đó, trên tab Windows Update, nhấp vào tùy chọn View update history.
  • Trên màn hình View update history, nhấp vào tùy chọn Uninstall updates.
  • Tìm KB4517389, KB4524147 và KB4517211 các bản cập nhật này rồi chọn chúng. Nếu bạn không thể tìm thấy tất cả chúng mà chỉ có một cái hiện diện, điều đó không sao. Đơn giản chỉ cần chọn đó.
  • Khi bạn đã hoàn tất việc đó, bạn cũng có thể chọn ngừng nhận các bản cập nhật để ngăn chặn bất kỳ sự cố nào như vậy bằng cách nhấp vào tùy chọn Tạm dừng cập nhật trong 7 ngày trên màn hình Windows Update trước đó.

# Tắt hỗ trợ tương thích

  • Mở Run và nhập gpedit.msc điều hướng đến: 
Computer Configuration > Windows Components > Application Compatibility
  • Sau đó, ở phía bên phải, nhấp đúp vào Turn off Application Compatibility Engine để mở các thuộc tính của nó.
  • Trên cửa sổ chính sách xuất hiện, hãy chọn tùy chọn Disabled – Apply – OK.

# Bỏ chặn VMware Workstation Pro bằng Reg File

  • Trước hết, hãy tải xuống tệp reg cần thiết từ đây.
  • Sau khi tải xuống, hãy tiếp tục và chạy tệp và khi được nhắc, hãy nhấp vào nút Yes.
  • VMware Workstation Pro của bạn sẽ hoạt động trở lại mà không gặp bất kỳ sự cố nào.

# Bỏ chặn VMware Workstation với ADK

  • Trước hết, hãy truy cập trang ADK của Windows bằng cách nhấp vào đây.
  • Tại đó, nhấp vào tùy chọn Download the Windows ADK để tải xuống trình cài đặt.
  • Khi trình cài đặt đã được tải xuống, hãy tiếp tục và chạy nó.
  • Đảm bảo rằng tùy chọn đầu tiên, tức là Install the Windows Assessment and Deployment Kit to this computer được chọn khi trình hướng dẫn cài đặt mở ra. Next.
  • Trên trang Select the features you want to install, hãy đảm bảo chọn Application Compatibility Tools ở trên cùng. Bạn có thể bỏ chọn mọi thứ khác nếu muốn.
  • Cuối cùng, nhấn Install để bắt đầu cài đặt.
  • Sau khi cài đặt xong, hãy mở Start Menu và tìm kiếm Compatibility Administrator. Mở nó ra.
  • Sau đó, mở rộng danh sách Applications trong System Database.
  • Từ danh sách các ứng dụng, tìm VMware Workstation Pro và chọn nó.
  • Ở phía bên phải, đối với các mục nhập có nội dung HARDBLOCK bên cạnh AppHelp, hãy nhấp chuột phải vào tệp .exe của nó và chọn Disable Entry từ menu thả xuống.
  • VMware bây giờ sẽ được bỏ chặn và bạn sẽ có thể sử dụng nó mà không gặp bất kỳ sự cố nào.

# Nâng cấp VMware Workstation Pro

Một cách để khắc phục là chỉ cần gỡ cài đặt VMware Workstation Pro khỏi hệ thống của bạn và sau đó tải xuống phiên bản mới nhất có sẵn. Sau khi tải xuống, bạn có thể cài đặt nó và sẽ không có vấn đề gì.
